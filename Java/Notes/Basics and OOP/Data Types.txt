Data Types:
============

Data Type is defined as the type of data that can be stored in a variable. 
It tells the Java Compiler how a user wants to use the data.
Java is not 100% object-oriented because it supports primitive data types.

As Java is a strongly typed language, you need to define the data type of the variable to use it and
you can not assign incompatible datatype otherwise the compiler will give you an error.

There are two types of data types in java.

1. Primitive Data Types
2. Reference or Non-Primitive Data Types


1) Primitive data types:
-----------------------------------------
Primitive data types store simple values directly in memory. 
Java provides eight primitive data types, each with a fixed size and range.

1. boolean: 
_____________
The boolean data type  is used with variable to store one of the values i.e. true and false
Use this data type for simple flags that track true/false conditions. 
This data type represents 1-bit of information, but its "size" isn't something that's precisely defined.

Example: boolean boolVar = true;

2. char: 
_____________
The char data type is used with variable to store a single 16-bit Unicode character. 
It has a minimum value of '\u0000' (or 0) and a maximum value of '\uffff' (or 65,535 inclusive).
It uses 2 bytes of space in memory to store each variable. 

Example: char charVar = 'a';

If we assign an integer value to a char type variable, the compiler will implicitly typecast the integer to char by converting it into its corresponding ASCII value.
So, if we assign char i = 97, then the variable i will hold the value a as the ASCII value of a is 97.


3. byte: 
_____________
The byte data type is used to store an 8-bit signed two's complement integer. 
It has a minimum value of -128 and a maximum value of 127 (inclusive). The byte data type can be useful for saving memory in large arrays, where the memory savings actually matters.
Its range lies between -128 to 127, if it exceeds either range the compiler will throw an error - "incompatible types: possible lossy conversion".

Example:  byte byteVar = 127; 

4. short: 
_____________
The short data type is a 16-bit signed two's complement integer. 
It has a minimum value of -32,768 and a maximum value of 32,767 (inclusive). 
We can use a short to save memory in large arrays, in situations where the memory savings actually matters.

Example: short shortVar = 4199;

5. int: 
_____________
The int data type is a 32-bit signed two's complement integer.
It has a minimum value of -2,147,483,648 and maximum value of 2,147,483,647 (inclusive). 

Example: int intVar = 401999;

6. long:
_____________
The long data type is a 64-bit two's complement integer. 
It has a minimum value of -9,223,372,036,854,775,808 and a maximum value of 9,223,372,036,854,775,807 (inclusive)
Use this data type when you need a range of values wider than those provided by int

Example: long longVar = 21474836471L;

7. float:
_____________
The float data type is a single-precision 32-bit floating point number (number with decimal points).
However, due to its limited precision (approximately 6-7 significant decimal digits), 
It is not suitable for applications where exact decimal values are required (for e.g. currencyValue).

Example: float floatVar1 = 3.14;
float floatVar2 = 14.19f;

8. double:
_____________
The double data type is a double-precision 64-bit floating point number (number with decimal points).
However, due to its limited precision (approximately 15-16 significant decimal digits), 
It is not suitable for applications where exact decimal values are required (for e.g. currencyValue).
For decimal values, this data type is generally the default choice. 

Example: float doubleVar1 = 3.14;
float doubleVar2 = 314.19d;

2) Reference or Non-Primitive Data Types
-----------------------------------------
Non-primitive data types store references (memory addresses) rather than actual values. 
They are created by users and include types like String, Class, Object, Interface, and Array.

1. String:
_____________
String data type is used to represents a sequence of characters.
Java strings are objects and are immutable.

Example: String stringVar ="CodeChamps";

Note: String cannot be modified after creation. Use StringBuilder class for heavy string manipulation.

2. Class:
_____________
A class is a user-defined blueprint or entity that defines variables and methods. 
It represents a type of object and forms the foundation of Object-Oriented Programming.
An Object is an instance of a class representing real-world entities. It has state (data), behavior (methods), and identity (unique reference)

Example:

class Student
{
	String name;
	int rollNo;

	Student(String name, int rollNo) {
        	this.name = name;
        	this.rollNo = rollNo;
    	}

}

public class Main {
    public static void main(String[] args) {
        Student stud = new Student("ABC", 21); // Student - class, stud - reference, new Student(..) - object)
        System.out.println("Name: " + stud.name);
        System.out.println("Age: " + stud.rollNo);
    }
}


3. Interface:
_____________
An interface in Java is a blueprint or entity that defines a set of methods a class must implement without providing full implementation details and variables.
It helps achieve abstraction by focusing on what a class should do rather than how it does it.
All variables in an interface are public, static, and final by default.
FYI - After Java 7 version - Java introduced default methods, static methods, private methods, 
private static methods, sealed interface and pattern matching with sealed interface.
We will see that Evolution later in Interface section.


Example:

interface StudentIFace
{
	final static int classNo = 10;
	void display();
}

class Student implements StudentIFace
{
	String name;
	int rollNo;

	Student(String name, int rollNo) {
        	this.name = name;
        	this.rollNo = rollNo;
  	}

	public void display(){
		System.out.println("Name: " + this.name);
  		System.out.println("Age: " + this.rollNo);
		System.out.println("Class: " + StudentIFace.classNo + "th");
	}

}

public class Main {
    public static void main(String[] args) {
        StudentIFace studIFace = new Student("ABC", 21); // StudentIFace - Interface, studIFace - reference, new Student(..) - object)
       	studIFace.display();
    }
}
 

4. Array:
_____________
An array stores fixed number of elements of the same type (primitive or non-primitive) in a single structure.
Each item in an array is called an element, and each element is accessed by its numerical index. 
Java arrays are objects, dynamically allocated, and indexed from 0.

Example:

public class Main {
    public static void main(String[] args) {
        int[] numbers = {1, 2, 3};
        String[] names = {"ABC", "PQR", "XYZ"};
        System.out.println("First number: " + numbers[0]);
        System.out.println("Second name: " + names[1]);
    }
}


4. Enum:
_____________
Enums are used to define a set of constants, providing a way to represent a fixed set of values.
Use enums when you have values that you know aren't going to change, like month days, days, colors, weeks, etc.

Example:
enum StudentsGrade {  
  FIRST,  
  SECOND,  
  THIRD  
}  

public class Main {
	public static void main(String ...args) {
		System.out.println("Highest Grade Name: " + StudentsGrade.FIRST);
		for (StudentsGrade myEnum : StudentsGrade.values()) {
  			System.out.println(myEnum);
		}

	}
}
