RandomAccessFile (java.io.RandomAccessFile class) object allows:
1) Establishing bidirectional file streams and hence file io (rw).
2) IO (rw) of primitive Java types.
3) Seek operation.


Key Characteristics:
-----------------------
~ Hybrid Nature: It handles both reading and writing simultaneously, so you don't need 
separate input and output streams.

~ Non-Sequential Access: You can skip directly to the middle or end of a file to modify or read data instantly without parsing everything before it.

~ Modes: It requires a mode specified at initialization: "r" for read-only, or "rw" for read and write.


Core Methods:
--------------
seek(long pos) - Moves the file pointer to the specified byte position from the beginning of the file.
getFilePointer() - Returns the current byte location of the file pointer.
length() - Returns the total size of the file in bytes.
write(int b) / read() -	Writes or reads a single byte at the current pointer position.
